## 📦 下载

### Windows 桌面应用
- **KINXKit-Desktop-0.1.0.exe** (74 MB) - 推荐

### CLI 工具
\```bash
npm install -g kinxkit
kinx create my-project
```

## 🚀 快速开始

### 使用桌面应用
1. 下载并安装 `KINXKit-Desktop-0.1.0.exe`
2. 启动应用，创建项目

### 使用 CLI
\```bash
kinx create my-project
kinx up
```
